# Changelog

All notable changes to the Laiqon Theme extension will be documented in this file.

## [1.0.0] - 2026-06-25

### Added
- **Laiqon** — new default dark theme
- **Laiqon Sapphire** — new blue-toned dark theme
- **Laiqon Ruby** — new red-toned dark theme

### Changed
- Refined color tokens across **Amethyst**, **Carbon**, **Graphite**, and **Light** themes
- Improved overall consistency of UI and syntax highlighting across the theme pack

### Removed
- **Laiqon Midnight** — replaced by the new Laiqon default theme
- **Laiqon Winter** — removed from the pack

## [0.0.2] - 2026-06-24

### Fixed
- Corrected repository URL in `package.json`

## [0.0.1] - 2026-06-24

### Added
- Initial release with Carbon, Amethyst, Graphite, Light, Midnight, and Winter themes
